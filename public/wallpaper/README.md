# צילומי מסך למדריך הרקע החי (/wallpaper)

הניחו כאן שלושה צילומי מסך מ-Lively Wallpaper, בשמות המדויקים האלה
(העמוד src/app/wallpaper/page.tsx מצפה להם):

1. add.png      — חלון "הוסף טפט" עם הדבקת קישור הספירה בשדה ה-URL.
2. display.png  — חלון "בחר תצוגה" עם התצוגה המקדימה של הספירה.
3. library.png  — הספרייה של Lively כשהספירה כבר מופיעה בה (התוצאה).

טיפ: רוחב ~1000px מספיק. אפשר להעלות דרך GitHub (Add file → Upload files)
לתוך התיקייה public/wallpaper/ בענף, או לשים את הקבצים כאן ולדחוף.
